# HNG-Stage-0-Todo-Card
This is a simple, responsive Todo Card component built as part of the HNG Internship Stage 0 task.

## 🚀 Features
- Displays task title and description
- Shows priority (High)
- Displays due date and time remaining
- Status indicator (Pending / Done)
- Checkbox to mark task as complete
- Tags (work, urgent)
- Edit and Delete buttons (dummy actions)

- ## 🛠️ Tech Used
- HTML
- CSS
- JavaScript

- ## 📌 Notes
- Built with accessibility and responsiveness in mind
- Time remaining is statically displayed for this stage
